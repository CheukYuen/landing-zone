📡 为什么新增 macro_daily？v1.3

DR007、YIELD_30Y、T5YIE 等指标在采购清单 v1.33 中被明确要求"保留日频原始 + 输出月均"。日频原始存 macro_daily，月均聚合写 macro_monthly。

macro_daily（日频原始）macro_monthly（月均聚合）
DR007 2026-02-01 = 1.82%DR007 2026-02 = 1.85%
DR007 2026-02-02 = 1.79%（月均值，source_tag=
DR007 2026-02-03 = 1.91%AGG_FROM_DAILY）

↑ 审计/复算可溯源至日频粒度；杠杆成本按日频逐日计提

🧮 derived_daily 存什么？v1.3

由已采购原始序列内部计算的合成特征。非采购项，在 indicator_catalog 中登记（category='internal_derived'）。

code公式用途
TERM_SPREAD_10Y_1YYIELD_10Y − CGB_1Y周期位置识别
TERM_SPREAD_30Y_10YYIELD_30Y − YIELD_10Y曲线形态监测
DR007_FUNDING_COSTDR007 日频计提杠杆资金成本
